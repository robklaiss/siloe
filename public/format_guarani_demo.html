formatGuarani() Function Demo

Function Implementation

/**
 * Format a number as Paraguayan Guaraní currency
 * @param {number} amount - The amount to format
 * @returns {string} Formatted currency string
 */
function formatGuarani(amount) {
    return new Intl.NumberFormat('es-PY', {
        style: 'currency',
        currency: 'PYG',
        minimumFractionDigits: 0,
        maximumFractionDigits: 0
    }).format(amount);
}

Usage

This function has been added to main.js and is available globally throughout the application.

Example usage in JavaScript:

// Format a price
const price = 25000;
const formattedPrice = formatGuarani(price);
element.textContent = formattedPrice;

Example usage in PHP views (with inline JS):

<span class="price"><?= htmlspecialchars("Precio: ") ?><span id="price-display"><?= $product['price'] ?></span></span>

<script>
    document.getElementById('price-display').textContent = formatGuarani();
</script>
